# Chaeco 组织

欢迎来到 Chaeco 组织！这是一个专注于开源软件开发的社区。

## 关于我们

Chaeco 是一个致力于创建高质量、可靠的开源项目的组织。我们主要用 TypeScript 开发，致力于为开发者社区提供实用的工具和库。

## 我们的使命

- 🚀 开发高质量的开源软件
- 💡 提供实用的开发工具和库
- 🌍 支持全球开发者社区

## 热门项目

### 路由和自动化

- **[route-wizard](https://github.com/chaeco/route-wizard)** - TypeScript 路由管理工具
- **[hoa-auto-router](https://github.com/chaeco/hoa-auto-router)** - 自动路由解决方案

### 认证和存储

- **[hoa-jwt-permission](https://github.com/chaeco/hoa-jwt-permission)** - JWT 权限管理库
- **[indexed-db-storage](https://github.com/chaeco/indexed-db-storage)** - IndexedDB 存储解决方案

### 实用工具

- **[logger](https://github.com/chaeco/logger)** - 灵活的日志库
- **[website](https://github.com/chaeco/website)** - 组织官方网站

## 如何参与

我们欢迎来自所有人的贡献！以下是参与方式：

1. **浏览我们的项目** - 查看我们的 GitHub 仓库
2. **提交 Issue** - 报告 bug 或提议新功能
3. **提交 Pull Request** - 贡献代码改进
4. **讨论** - 参与社区讨论

## 联系我们

- 📧 通过 GitHub Issues 联系
- 💬 在我们的仓库讨论区参与讨论

## 许可证

我们的项目采用开源许可证。具体信息请查看各个项目的许可证文件。
