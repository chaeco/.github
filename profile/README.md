# 欢迎来到 Chaeco 组织 👋

这是一个专注于开源软件开发的社区。

## 关于我们

Chaeco 是一个致力于创建**高质量、可靠的开源项目**的组织。我们主要用 **TypeScript** 开发，致力于为开发者社区提供实用的工具和库。

## 🎯 我们的使命

- 🚀 开发高质量的开源软件
- 💡 提供实用的开发工具和库  
- 🌍 支持全球开发者社区
- 📚 分享知识和最佳实践

## 🤝 如何参与

我们热烈欢迎来自社区的贡献！参与方式：

✨ **浏览我们的项目** - 发现有趣的开源项目  
🐛 **提交 Issue** - 报告 bug 或提议新功能  
🔧 **提交 Pull Request** - 贡献代码改进  
💬 **参与讨论** - 在社区讨论区分享想法  

### 贡献指南

详见各个项目的 `CONTRIBUTING.md` 文件了解具体步骤。
